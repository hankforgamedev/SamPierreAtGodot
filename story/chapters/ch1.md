[narrator none] 虎寨城，地鐵下城站。深夜，末班車前。

[narrator none] 山姆．皮耶爾從警局大門走了出來，無奈地嘆了口氣，然後隨手打開生鏽的煙盒，點了一根卡斯特。照常加班完後，這是他唯一能做的小小消遣，讓尼古丁重新整理他的思緒。

[narrator none sfx:mouse_click ]

年久失修的長椅上，皮耶爾坐著。嘎吱——感覺再過一秒長椅就要解體了。他又點了一根卡斯特，眼光凝聚在月台的柱子上。

下面的空氣比街上還糟，霉味、鼠屍臭、尿騷味佈滿整個空間，跟警局的停屍間一樣臭。月台上一個人都沒有。

[narrator none] 這裡充斥著粗俗下流的塗鴉與幾則爛笑話：

[sam left]
「為甚麼列寧喜歡穿皮鞋，史達林卻總是穿靴子？」

「因為列寧時代，俄羅斯的血水只淹到腳踝。」

[narrator none speed:slow] 皮耶爾其實覺得沒有很好笑，他不喜歡太寫實的笑話。

[rat right sfx:horse-walking-sound] 「嘿……山姆，對嗎？好久不見，呵。」

[sam left] 「我們見過嗎？」

[narrator none] 皮耶爾疑問道，他認識的人大多都疏遠到連在路上跟他裝熟打招呼都不會。在這個時間這種鳥地方怎麼可能會遇到正常人。

[rat right] 「你怎麼這麼無情？我可是你老媽的情人，哈哈……」

[narrator none sfx:horse-walking-sound speed:fast] 月台昏黃的燈光下，那頭金色的邋遢捲髮巧妙融入，原來是上禮拜在局裡，因為騷擾山姆，被獄卒痛扁的癮君子。那人加快腳步走到皮耶爾旁邊。他穿著灰色連帽外套、黑色搖滾 T 恤，牛仔垮褲配皮靴。

[sam left] 「原來是你啊，老鼠。」

[rat right] 「什麼老鼠，你用那張臭嘴親你老媽嗎？真巧，我也用我這張臭嘴親你老媽！」

[rat right] 「最近過得如何啊老兄，有沒有宰了你那肥豬上司？」

[sam left] 「正在考慮了，別著急。」

[narrator none] 老鼠在皮耶爾身旁坐下，也不管這個孤獨的山姆多不自在。

[rat right] 「老兄，你過得快樂嗎？」

[sam left] 「多謝你了，老鼠，我特別煩悶。」

[narrator none speed:fast] 皮耶爾看都不看他一眼，繼續抽自己的香菸。

[rat right sfx:FAHHH] 「別再叫我老鼠！」

[narrator none speed:fast] 老鼠突然大吼，又愣了一下，若無其事繼續說話。

[rat right] 「說說看嘛，還是其實你抽菸也是為了像那些文青一樣耍帥嗎？」

[narrator none] 皮耶爾不發一語，任由冬天的冷空氣凝住寂靜。

[rat right] 「我看倒不像這樣，哈……你可真是個苦悶的人，那天我就看出來了。我特別會看人。」

[rat right] 「那些怨念還有烏煙瘴氣，整個籠罩在你身上，我是說真的啦……幹，而且啊，死神在你頭上晃來晃去餒。您是貨真價實的憂鬱小子啊！非常憂鬱！」

[rat right sfx:canned_laughter_longer] 「誠心建議你趕快去市中心跳一跳樓，早死早超生，順道壓壓房價。」

[narrator none] 老鼠似乎被自己的笑話逗樂，突然暈頭轉向，跌到地上。

皮耶爾想起老鼠上次也給過自己這樣的建議。

[rat right] 「你知道嗎？這個月石油漲價，我們的古柯鹼也漲價了。有個戴面具的雞巴步槍人把國產的藥頭都射死了，害我只能買垃圾進口貨，操。」

[narrator none] 他扶著長椅，緩緩爬起來。

[rat right] 「說到這個，借些錢花花吧，我把錢都拿去買粉吸掉了，沒錢吃起司漢堡了。不然這樣吧，借我一百，我買兩個漢堡——一個你看我吃——一個我吃給你看——」

[rat right choices sfx:canned_laughter_shorter]
老鼠邊笑邊說著，突然有些喘不過氣。

[narrator none label:PIERRE_STANDS] 皮耶爾察覺到老鼠那遲緩兒一般的怪異行徑，但或許對毒蟲而言，這樣才是常態。他站了起來，想要趕緊從這場鬧劇抽身，但老鼠用不停抽搐的手一把抓住了他。

[sam left speed:slow] 「滾開。」

[narrator sfx:hitHurt none] 他用力一揮，把老鼠甩開，然後退到身邊的柱子旁。

[narrator none speed:fast fx:flash_white]
老鼠從皮靴內側抽出一把十五公分的蝴蝶刀，衝向皮耶爾。

[narrator none] 皮耶爾看到那刀就愣住了。「要是我死了怎麼辦？」他腦中的思緒開始爆炸。

[inner none speed:fast fx:glitch_spike inner]
「老媽怎麼辦？誰來養她？還有李先生，要是知道我會這麼早死，我就會先送他下地獄了，不，不只他，還有那些卑劣的同事、嘻皮笑臉的鄰居、冷漠疏離的人群，還有父親。」

「最可悲的是，在我死之前，我一刻都沒有活成我想要的樣子過，我一輩子都是看著別人噁心的嘴臉做事的，完全不敢抗命。」

「操！一次就好，就算只有一次也好，我想活著，真正的活著，而那就是現在。」

[narrator none sfx:hitHurt_var1 speed:slow fx:shake] 不過有點太遲了。皮耶爾反應過來掏槍時，他的下腹已被蝴蝶刀刺中，正好是有舊傷的部位。

[narrator none speed:fast] 皮耶爾整個人被老鼠按在牆上，劇烈的疼痛讓他快要昏死過去，但他知道他不能閉眼。他仍試著擠出最後一點力氣開槍。

[narrator none sfx:gunshot speed:fast fx:shake]
砰！子彈只打中老鼠的肩膀。

[narrator none] 不過這小王八蛋先前嗑了好一包古柯鹼，根本感覺不到疼痛，只是笑了起來。蝴蝶刀繼續在皮耶爾的腸子間悠閒地散步。

似乎萬念俱灰了。

[narrator none speed:fast sfx:explosion_stuffy fx:shake] 好在皮耶爾終於等到了他的腎上腺素，他一記左鉤拳打在了老鼠的下巴上。

[narrator none speed:fast] 「焯！」老鼠咬到舌頭，向後蹣跚打跌幾步。

[narrator none sfx:gunshot fx:shake] 皮耶爾又朝他的胸口開了一槍，射穿了他的肺。老鼠摔了一大跤跌坐在地上。

[narrator none]
「嘶……嘶……」那氣球洩氣似的聲音在月台上迴盪。

[narrator none fx:flash_white] 隧道深處傳來一束微弱的光，皮耶爾笑了。

[narrator none speed:slow] 他把老鼠抓了起來，朝月台邊拖了過去。這可憐的鼠輩只得用肺裡僅存的一些空氣發出些聽起來像求饒的聲響。

但皮耶爾毫不理會，他早已失去理智。

[narrator none] 那束光越來越近，車輪和軌道發出的尖嘯聲愈發刺耳，老鼠愣在月台邊四肢發軟動彈不得。

[narrator none sfx:tom_screaming speed:fast fx:flash_white,glitch_spike,train]
皮耶爾硬生生把他拽了起來，朝軌道的方向用力一推。「唰，噗吱！」末班車進站，可憐的鼠輩就像果凍一樣被輾碎。

[narrator none] 鮮血濺到皮耶爾的臉上、衣服上，朱紅的顏料在白襯衫上渲染了好一幅抽象畫。

[narrator none] （皮耶爾靠在柱子上，緩緩滑坐到地上）
……某種程度上，他很快樂。至少明天沒有擁擠的地鐵，也沒有痛苦的工作等著他。

[narrator none speed:slow] 他知道，他大概沒辦法搭上這班車回家了。

[narrator none ] 一名旅客穿過閘門，走上地鐵，回頭撇了一眼，然後轉頭繼續看他的晚報。下城區的流浪漢死在路邊，怪嗎？不過稀鬆平常。沒人想多管閒事。

[narrator none speed:fast] 地鐵停了半分鐘就走了，這個世界到最後依然這麼冷漠，但皮耶爾覺得無所謂。

[narrator none fx:flash_black]
除了地鐵，他感覺有東西正在離他而去，或許能稱之為靈魂的東西。

[narrator none] 他的身體越來越輕。似乎有人正把他的靈魂從他體內抽出來，像點著的菸一樣，一口一口，輕輕地燃著吐著，越來越短。

[narrator none speed:slow fx:flash_white,flash_black]
漸漸的，他失去感覺了。最後一口，他眼前被點點白光填滿，然後是一片黑。

[narrator none speed:fast sfx:tv_crash_noise]
最後什麼都沒有了。
