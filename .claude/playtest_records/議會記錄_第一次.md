## 議會技術備忘

### `assert_dict().contains_key()` 不存在

GDUnit4 v6.1.3 的 `GdUnitDictionaryAssertImpl` 沒有 `contains_key()`（單數）。改用：

```gdscript
assert_bool(dict.has(key)).is_true()
```

### `_ambient_label` 在測試環境是 null

`_ambient_label` 在 `_ready()` 裡建立，但測試直接 `new()` 不走 scene tree。解法：

```gdscript
func _attach_label(level: AsciiLevelBase) -> Label:
    var lbl := auto_free(Label.new()) as Label
    level._ambient_label = lbl
    return lbl
```

### `assert_vector2i()` 不存在

改用 `assert_that()` 泛型斷言，支援 Vector2i 比較。
